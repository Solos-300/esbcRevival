ESBC Revival PC-only Undisputed mod
ESBC REVIVAL

A revival built for PC — gameplay, roster expansion, kit modernisation, and a full UI refresh.

ESBC Revival is a structured mod project for Undisputed on PC, delivered in phases. Each phase focuses on a specific pillar: in-ring gameplay, new fighters, kit authenticity, and a redesigned main menu experience.

Explore phases ↓ Credits

Project Focus

OVERVIEW

Built to feel sharper, faster, and more alive — while keeping a consistent identity across gameplay, fighters, kits, and menus.

  • Gameplay + AI behaviour improvements
  • New camera feel and angles
  • Roster expansion using new fighters + CABs
  • Updated kits for select marquee boxers
  • Main menu artwork + interface overhaul

Phase 1

Gameplay Overhaul

IN-RING

The foundation of ESBC Revival. Phase 1 focuses on how the game feels moment-to-moment: pace, decision-making, stamina, and camera presentation.

  • Gameplay and AI updates to improve reactions, pressure, and overall ring IQ.
  • New camera updates for clearer reads and a more broadcast-like feel.
  • Boxer speed overhauls to balance movement and make styles stand out.
  • New boxing gameplay angles to change the feel of exchanges and spacing.
  • Stamina mechanic overhaul to create more realistic pacing over rounds.

Phase 2

Brand New Boxers

ROSTER

Phase 2 expands the experience beyond what’s included by default, focusing on fresh matchups, deeper variety, and a stronger sense of a living roster.

  • Replacing licensed boxers with new fighters to refresh the lineup.
  • Custom CABs (Created Boxers) added to increase roster depth and options.
  • New kits for every replaced licensed boxer to keep presentation consistent.
  • Extra kits for select CAB fighters for variety and identity.

Phase 3

Licensed Boxer Kit Updates

KITS

Phase 3 is about authenticity: updating marquee fighters to better reflect their most recent real-world looks, so the biggest names feel current.

  • Replacing select kits with their most recent designs and details.
  • Examples include Jesse “Bam” Rodriguez and Oleksandr Usyk.
  • Focus on accuracy, colour matching, and cleaner presentation in-game.

Phase 4

Main Menu Overhaul

UI

Phase 4 refreshes the entire first impression: menus, textures, and presentation elements so ESBC Revival feels like its own complete package.

  • Editing menu colours and interface textures across the main UI.
  • Replacing the start screen with ESBC Revival branding.
  • Replacing boxer images on menu buttons for a consistent look.
  • Updating the background to match ESBC Revival’s identity.
